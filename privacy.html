Privacy Policy for AInform Daily Post

Last Updated: April 29, 2026

This Privacy Policy describes Our policies on the collection, use, and disclosure of Your information when You use the AInform Daily Post application (the "Service"). We strictly adhere to the TikTok Developer Terms of Service.

1. Data Collection and Use

AInform does not collect, store, or share any personal data.

  • TikTok API Data: Our Service uses the TikTok Content Posting API to upload and manage video content. We only access the specific data required to perform these actions (e.g., uploading videos, adding captions).
  • No Retention: We do not store TikTok user data or video media on our servers. Any data accessed via the API is processed in real-time and is not retained.
  • Usage Data: We may track basic diagnostic "Usage Data" (like IP addresses) solely for security, but this is never linked to your TikTok identity.

2. Use of Your Personal Data

We use information accessed via the TikTok API exclusively to:

  • Provide and maintain the Service (uploading content as directed).
  • Ensure the security of the connection between n8n and TikTok.
  • Prohibition on Selling Data: We will not sell, rent, or lease any data obtained through the TikTok API to third parties.

3. Account Access and Security

Access is managed through secure OAuth tokens via your self-hosted n8n instance. You can revoke access at any time through your TikTok security settings.
